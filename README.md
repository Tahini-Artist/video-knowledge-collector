# Video Knowledge Collector

视频知识收集器 - 从视频教程中提取价值,构建个人知识库。

## 功能特点

- 🔗 **链接解析** - 支持小红书、抖音、B站链接
- 📝 **完整知识点提取** - 提取视频中所有知识点，不遗漏
- ✅ **个性化行动建议** - 基于你的身份、业务、目标生成
- 🔗 **主题关联** - 按主题分析视频素材之间的相关性
- 📝 **Obsidian 归档** - 自动写入 Obsidian 知识库

## 适用场景

- 创业者收集视频教程内容
- 构建个人知识库
- 从视频中提取行动建议
- 分析视频素材主题关联

## 安装方法

### 方法 1:直接安装

将此 Skill 复制到你的 OpenClaw skills 目录:

```bash
cp -r video-knowledge-collector ~/.qclaw/skills/
```

### 方法 2:从 .skill 文件安装

```bash
# 在 OpenClaw 中对 Agent 说:
安装 video-knowledge-collector.skill
```

## 使用方法

### 首次使用（必填）

**Step 1：提供你的身份信息**

```
我的身份是：独立创业者
我当前在做：AI 写作工具
我的目标是：6 个月 1 万付费用户
```

这 3 个信息用于生成**个性化行动建议**，后续无需重复填写。

**Step 2：配置 Obsidian 库路径（可选）**

```
我的 Obsidian 库路径是：/Users/你的用户名/Obsidian/我的知识库
```

### 日常使用

**分析单个视频：**
```
帮我分析这个视频：https://www.bilibili.com/video/BV1xxx
```

**批量整理：**
```
整理这几个视频到我的知识库：
- https://www.bilibili.com/video/BV1xxx
- https://www.xiaohongshu.com/discovery/item/xxx
```

**查看知识网络：**
```
我收集了哪些关于“流量获取”的视频？
```

**更新个人信息：**
```
更新我的目标：现在是 3 个月做到 5 万粉丝
```

## 笔记模板

每条视频素材会生成如下格式的笔记:

```markdown
---
title: 视频标题
source: 平台
url: 视频链接
author: 博主名称
collected: 收集时间
topic: 主题分类
tags:
  - 视频素材
  - 平台
  - 主题
---
```
