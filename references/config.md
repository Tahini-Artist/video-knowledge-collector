## 默认设置

### 视频平台优先级

当视频同时在多个平台发布时，优先使用：

1. B站（字幕较完整）
2. 小红书
3. 抖音

### 知识点数量

每个视频默认提取 3-7 个核心知识点。

### 行动建议分类

- 立即执行：今天可完成
- 短期计划：1-7 天完成
- 长期探索：持续投入

---

## 首次使用说明

1. 告诉 Agent 你的 Obsidian 库路径
2. Agent 会自动创建目录结构
3. 创建完成后即可开始使用

**示例：**
```
我的 Obsidian 库路径是：/Users/mingyuewu/Documents/Obsidian/我的知识库
```
