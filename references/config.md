# 用户配置

## Obsidian 知识库路径

> ⚠️ 首次使用必须填写，用于自动写入视频笔记

```yaml
# 请在此填写你的 Obsidian 库路径
# macOS 示例：
# obsidian_path: /Users/你的用户名/Obsidian/我的知识库
# Windows 示例：
# obsidian_path: C:\Users\你的用户名\Documents\Obsidian\我的知识库

obsidian_path: [待填写]
```

## 用户信息

> ⚠️ 首次使用必须填写，用于生成个性化行动建议

详细用户画像保存在：`references/user-profile.md`

**必填项：**
- 身份：你的职业/角色
- 业务情况：你当前在做什么业务/项目
- 目标：你希望达成什么目标
- Obsidian 路径：你的知识库本地路径

## 默认设置

### 视频平台优先级

当视频同时在多个平台发布时，优先使用：

1. B站（字幕较完整）
2. 小红书
3. 抖音

### 知识点提取

- **完整性优先**：提取视频中所有知识点，不限制数量
- 保留原始信息：数字、步骤、工具名称、案例
- 标注时间点：便于回看

### 行动建议分类

- 📌 立即执行：今天可完成（30 分钟内）
- 📅 短期计划：1-7 天完成
- 🎯 长期探索：持续投入
- ❌ 不适用项：说明原因并给出替代方案

### 文件命名规则

```
格式：{平台}-{视频标题}.md

示例：
- B站-如何用AI提升写作效率.md
- 小红书-独立开发者必装的6个工具.md
- 抖音-一天涨粉1000的实操方法.md

注意：
- 标题过长时截取前 50 字符
- 移除文件名中的非法字符（/ \ : * ? " < > |）
```

## 首次使用说明

### 一键初始化

```
请初始化视频知识收集器，我的信息如下：
- 身份：独立创业者
- 业务情况：AI 写作工具
- 目标：6 个月 1 万付费用户
- Obsidian 路径：/Users/mingyuewu/Obsidian/我的知识库
```

### 或分步配置

```
我的身份是：独立创业者
我当前在做：AI 写作工具
我的目标是：6 个月 1 万付费用户
我的 Obsidian 库路径是：/Users/mingyuewu/Obsidian/我的知识库
```

### 第三步：开始使用

发送视频链接即可开始分析，笔记会自动写入 Obsidian。
